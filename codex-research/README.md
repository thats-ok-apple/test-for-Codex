# CodeX 每日教程与案例调研（Github / 小红书 / B站 / YouTube / X）

这个目录用于“每天一点”沉淀 CodeX 使用方法、实战案例和可复用教程。

## 工作流（建议每天 15-30 分钟）

1. 在各平台按关键词检索（中英文都搜）。
2. 把高价值内容追加到 `daily-log.md`。
3. 运行 `python3 codex-research/scripts/generate_weekly_digest.py` 生成周报草稿。
4. 提交到 GitHub。

## 推荐检索关键词

- `CodeX 教程`
- `CodeX 实战`
- `OpenAI Codex workflow`
- `Codex agent coding`
- `Codex prompt 技巧`
- `Codex code review`

## 平台维度记录建议

- **来源平台**：GitHub / 小红书 / 哔哩哔哩 / YouTube / X / 其他
- **标题与链接**
- **内容类型**：入门 / 进阶 / 案例 / 踩坑
- **核心方法**（3 条以内）
- **可复用程度**：高 / 中 / 低
- **可信度**：高（官方/作者）/ 中（转述）/ 低（无验证）

## 发布建议

- 每日：更新 `daily-log.md`
- 每周：更新 `weekly-digest.md`
- 每月：整理 `monthly-best-practices.md`
